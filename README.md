# STM32 Embedded Systems — 8 Week Learning Program

A structured 8-week self-study program in embedded systems using the WeAct STM32F405RGT6 Blackpill board, culminating in a TinyML anomaly detection project deployable to an ESP32.

## Board & Hardware

- STM32F405RGT6 (WeAct Blackpill) — ARM Cortex-M4, 168 MHz, 1MB Flash, 192KB SRAM
- ESP32 (capstone deployment target)
- Development environment: STM32CubeIDE on Windows

## Program Structure

| Week | Topic |
|------|-------|
| 1 | Architecture, memory map, bare-metal GPIO |
| 2 | UART and serial communication |
| 3 | ADC and analog sensors |
| 4 | Timers and interrupts |
| 5 | I2C and SPI protocols |
| 6 | DMA and data pipelines |
| 7 | Machine learning fundamentals and TinyML training |
| 8 | Capstone — end-to-end anomaly detection system |

## Goal

Build a strong practical foundation in STM32 embedded development from bare-metal register programming through to communication protocols and DMA. The final project combines the embedded pipeline with a TinyML model deployed to an ESP32, with a live WiFi dashboard displaying sensor data and anomaly classification in real time.
